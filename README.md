# 📈 Zerodha Clone

A full-stack Zerodha Clone built using the MERN Stack. This project replicates the core features of the Zerodha trading platform with a modern and responsive user interface.

## 🚀 Features

- 🔐 User Authentication
- 📊 Interactive Dashboard
- 📈 Holdings & Positions
- 💼 Portfolio Management
- 📱 Responsive Design
- ⚡ Fast React Frontend
- 🗄️ MongoDB Database
- 🔗 REST API using Express.js

## 🛠️ Tech Stack

### Frontend
- React.js
- HTML5
- CSS3
- JavaScript
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/sandeepksaini17/ZerodhaClone.git
```

### Go to Project Folder

```bash
cd ZerodhaClone
```

### Install Dependencies

```bash
npm install
```

### Start Backend

```bash
cd backend
npm install
npm start
```

### Start Frontend

```bash
cd frontend
npm install
npm start
```

### Start Dashboard

```bash
cd dashboard
npm install
npm start
```
